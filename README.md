# Goblin Scan 🧌
GoblinScan is a lightweight Python-based cybersecurity toolkit featuring an interactive menu for quick Nmap and Gobuster scans. Designed for simplicity and portability, it provides fast access to essential reconnaissance commands, making it ideal for beginners and small pentesting workflows.
Features

## Nmap Scans

    Fast Scan — quick service/version detection

    Full Scan — aggressive scan with detailed output

    Ports Scan — scan specific ports chosen by the user

    OS & Versions Scan — detect operating system and service versions

## Gobuster Scans

    Directory brute‑forcing

    Custom wordlist support

## Interactive Menu

    Clean, simple interface

    Submenus for better organization

    Goblin‑themed ASCII banner

## Requirements

    Python 3

    Nmap

    Gobuster
    
## Installation
```bash
git clone https://github.com/rsilva-25/GoblinScan.git
cd GoblinScan
python goblinscan.py

```
